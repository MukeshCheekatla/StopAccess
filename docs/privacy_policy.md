# Privacy Policy
> Updated: 30 Mar 2026

## Summary

FocusGate is designed to keep product state local and only talk to NextDNS when the user explicitly connects their own account.

## On-Device Data

- rules
- schedules
- focus state
- insights
- logs
- NextDNS profile ID and API key

## Network Use

FocusGate talks to NextDNS only to manage the user’s own profile configuration.

## Product Principle

No FocusGate-operated analytics backend is assumed in the current product model.
